There is always a certain confusion that arises while installing Java in the linux operating system. This is due to the vagueness behind the understanding of following questions, What is JDK and JRE? and, Are both of them the same?. Among JDK and JRE, which is needed to run java programs in the system?.

The quick answer to the question is, JDK alone is sufficient and to confuse you a bit, JRE alone is also able to run the java programs :). Then why there are 2 separate entities,

  1. JDK or Java Development Kit is a package which consists of both compile-time and run-time capabilities required for the execution of a Java program. In other words, JDK is a software development kit which compiles the program and also has a Java virtual machine to run the program.
  2. However, JRE or Java Runtime Environment cannot help in development and compilation of the program but has the Java virtual machine capability to run the Java program.

Now we have some answers in hand and moving forward, there is a question that if the programmer develops and has the compiled object code, he/she doesn’t need JDK anymore. This applies to only certain cases as some of the applications require JDK even though the code is already compiled. This is due to the requirement of certain modules required while running the application in real time. One such example is the requirement of servlet generator packages while executing the web application.

The use of JRE is optimal, but should be confident that the application doesn’t need JDK anywhere.

Installation commands:

Infrastructure Info:
  • Linux - Ubuntu 18
  • 16G Memory

JDK can be installaed using the following commands,

sudo apt-get update
sudo apt install default-jdk
java -version

JRE can be installed using the command,

sudo apt install default-jre

Another point where developers go into confusion is setting up the JAVA_HOME. The JAVA_HOME is an environment variable used by some of the standard JAVA projects like Kafka to locate the java libraries for execution.

The following commands are used to set up the JAVA_HOME variable,

  1. Locate the java installation using the command:
update-alternatives --config java
  1. Specify the location in /etc/environment file:
sudo vim /etc/environment
JAVA_HOME="/usr/lib/jvm/java-11-openjdk-amd64/bin/java"
source /etc/environment
  1. Confirm the set-up using,
echo $JAVA_HOME
